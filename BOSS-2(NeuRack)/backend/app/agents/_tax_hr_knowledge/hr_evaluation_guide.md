# 인사평가서 작성 기준 자료

## 법적 근거

- 근로기준법 §23: 정당한 이유 없는 해고·징계 금지 → 평가 기록이 인사 결정의 객관적 근거
- 근로기준법 §76조의2~3: 직장 내 괴롭힘 방지 → 평가 과정의 공정성·투명성 의무

## 평가 유형별 주기

| 유형      | 주기               | 비고                            |
| --------- | ------------------ | ------------------------------- |
| 연간 평가 | 매년 12월 또는 1월 | 연봉 협상·승진 반영             |
| 반기 평가 | 6월, 12월          | 상반기·하반기 각 1회            |
| 분기 평가 | 3·6·9·12월         | 단기 목표 관리                  |
| 수시 평가 | 수시               | 특별 프로젝트, 시용기간 종료 전 |

## 권장 평가 지표 (5점 척도)

### 공통 지표 (업종 무관)

| 지표        | 배점 | 측정 기준                       |
| ----------- | ---- | ------------------------------- |
| 업무 성과   | 5점  | 목표 달성률, 매출·생산성 기여도 |
| 업무 태도   | 5점  | 근태, 복장, 기본 예절           |
| 고객 응대   | 5점  | 고객 만족도, 컴플레인 처리      |
| 팀워크·협업 | 5점  | 동료와의 협력, 갈등 관리        |
| 성장 의지   | 5점  | 자기계발, 새 업무 습득          |

### 업종별 추가 지표 예시 (사업장 업종에 맞게 선택 적용)

| 업종        | 추가 지표 예시                                      |
| ----------- | --------------------------------------------------- |
| 음식·카페   | 위생·청결 (식품위생법 준수), 제품 품질, 스피드·효율 |
| 소매·유통   | 재고 관리 정확도, 진열·매장 관리, 판매 목표 달성률  |
| 미용·뷰티   | 기술 숙련도, 예약 이행률, 고객 재방문율             |
| 학원·교육   | 수업 준비도, 학생 만족도, 출결 관리                 |
| 물류·배달   | 배송 정확도, 사고율, 처리량                         |
| 서비스·기타 | 업무 숙련도, 응대 속도, 고객 만족도                 |

_LLM은 프로필의 업종(business_type)을 참고해 해당 업종에 맞는 추가 지표를 선택하세요._

## 5점 척도 기준

| 점수 | 등급     | 기준                        |
| ---- | -------- | --------------------------- |
| 5    | S (탁월) | 기대 수준을 현저히 초과     |
| 4    | A (우수) | 기대 수준을 초과            |
| 3    | B (보통) | 기대 수준 충족              |
| 2    | C (미흡) | 기대 수준 미달, 개선 필요   |
| 1    | D (불량) | 현저히 미달, 즉각 개선 요구 |

## 종합 등급 산출

| 평균 점수 | 종합 등급 | 조치 권고                                    |
| --------- | --------- | -------------------------------------------- |
| 4.5 이상  | S         | 포상, 승진 우선 검토                         |
| 3.5~4.4   | A         | 긍정적 피드백, 역할 확대                     |
| 2.5~3.4   | B         | 현 직무 유지, 약점 보완 지도                 |
| 1.5~2.4   | C         | 개선 계획서 작성, 재평가 3개월 후            |
| 1.4 미만  | D         | 인사위원회 회부, 해고 검토 시 법적 절차 준수 |

## 작성 주의사항

- 평가자 서명·날짜 필수 (법적 증빙)
- 평가 결과를 피평가자에게 반드시 공지 (근로기준법 §23 관련 판례)
- 인종·성별·종교·나이 등 차별적 사유 기재 금지 (남녀고용평등법 §8)
- 보관: 3년 이상 (근로기준법 §42)
