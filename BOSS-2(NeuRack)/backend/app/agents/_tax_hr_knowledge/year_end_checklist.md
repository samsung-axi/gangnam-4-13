# 연말정산 체크리스트 기준 자료 (근로소득자용)

## 법적 근거

- 소득세법 §137 (근로소득 연말정산), §140 (소득공제신고서 제출)
- 국세청 연말정산 간소화 서비스: 매년 1월 15일 개통

## 일정

| 단계               | 시기             | 내용                                  |
| ------------------ | ---------------- | ------------------------------------- |
| 간소화 서비스 개통 | 1월 15일         | 국세청 홈택스에서 공제 자료 조회      |
| 소득공제 자료 제출 | 1월 20일까지     | 회사에 간소화 자료 + 별도 영수증 제출 |
| 연말정산 정산      | 2월 급여 지급 시 | 환급 또는 추가 납부                   |
| 지급조서 제출      | 3월 10일         | 회사가 국세청에 제출                  |

## 소득공제 항목별 체크리스트

### 인적공제 (소득세법 §50)

- [ ] 기본공제 대상자 확인 (본인·배우자·부양가족)
  - 배우자: 연 소득 100만 원 이하
  - 부양가족(직계존속): 만 60세 이상, 연 소득 100만 원 이하
  - 부양가족(자녀): 만 20세 이하, 연 소득 100만 원 이하
- [ ] 추가공제: 경로우대(70세 이상 100만 원), 장애인(200만 원), 부녀자(50만 원), 한부모(100만 원)

### 연금보험료공제 (소득세법 §51의3)

- [ ] 국민연금 납부 확인서 (간소화 자동 반영)
- [ ] 공무원·군인·사학연금 납부액

### 특별소득공제

- [ ] 건강보험료·고용보험료 납부액 (간소화 자동 반영)
- [ ] 주택임차차입금 원리금 상환액 (근로자 주택 마련 저축 포함)
- [ ] 장기주택저당차입금 이자 상환액

### 신용카드 등 소득공제 (조특법 §126의2)

- [ ] 신용카드 사용액 (총급여의 25% 초과분의 15%)
- [ ] 체크카드·현금영수증 사용액 (초과분의 30%)
- [ ] 전통시장·대중교통 사용액 (초과분의 40~80%)
- 공제 한도: 총급여 7천만 원 이하 → 300만 원

### 세액공제 항목

#### 자녀세액공제 (소득세법 §59의2)

- [ ] 기본: 자녀 1명 15만 원, 2명 35만 원, 3명 이상 35만 원 + 1명당 30만 원 추가
- [ ] 출산·입양: 첫째 30만 원, 둘째 50만 원, 셋째 이상 70만 원

#### 연금계좌세액공제 (소득세법 §59의3)

- [ ] IRP·연금저축 납입액 확인 (총급여 5500만 원 이하: 16.5%, 초과: 13.2%)
- 한도: IRP 포함 연 900만 원

#### 보험료세액공제 (소득세법 §59의4)

- [ ] 보장성 보험료: 연 100만 원 한도, 12% 세액공제
- [ ] 장애인 전용 보험료: 연 100만 원 한도, 15% 세액공제

#### 의료비세액공제 (소득세법 §59의4)

- [ ] 본인·65세 이상·장애인: 전액 (한도 없음), 15%
- [ ] 그 외 부양가족: 총급여의 3% 초과분, 15% (한도 700만 원)
- [ ] 산후조리원: 200만 원 한도

#### 교육비세액공제 (소득세법 §59의4)

- [ ] 취학 전 아동·초중고: 1인당 연 300만 원 한도, 15%
- [ ] 대학생: 1인당 연 900만 원 한도, 15%
- [ ] 본인 교육비: 전액, 15% (대학원 포함)

#### 기부금세액공제 (소득세법 §59의4)

- [ ] 법정 기부금 (국가·지자체·국방헌금 등): 15~30%
- [ ] 지정 기부금 (종교단체 포함): 10~30%
- 기부금 영수증 원본 보관 필수

### 별도 제출 서류 (간소화에 미포함)

- [ ] 월세 세액공제 (연 750만 원 한도, 15~17%) — 임대차계약서 + 계좌이체 확인서
- [ ] 안경·렌즈 구매 영수증 (의료비 포함, 1인당 50만 원 한도)
- [ ] 중고생 교복 구매 영수증 (1인당 50만 원 한도)
- [ ] 해외 교육기관 교육비 영수증

## 사업주 유의사항

- 근로자가 제출 안 하면 기본공제만 적용 → 근로자 불이익 발생
- 허위 자료 제출 시 가산세 + 추징 (소득세법 §81)
- 중도 퇴사자: 퇴사 월 급여 지급 시 연말정산 실시
