# 통신판매업 신고서 — 메타 정보

## 관할 기관

- 사업장 소재지 관할 **시·군·구청** (지역 담당 부서)
- 온라인 신청: https://www.gov.kr (정부24)

## 처리 기한

- **7일** (영업일 기준)
- due_label: `통신판매업 신고 처리 기한`

## 근거 법령

- 전자상거래 등에서의 소비자보호에 관한 법률 §12 (통신판매업자의 신고)
- 동법 시행규칙 [별지 제1호서식]
- 위반 시 제재: 시정권고, 영업정지, 과태료 최대 3,000만원 (§32·§45)

## 신고 대상

- 전자상거래·통신판매를 업(業)으로 하는 자
- 연간 거래 횟수 50회 미만 또는 연간 매출 1,200만원 미만은 면제 가능

## 첨부 서류

- [ ] 구매안전서비스 이용 확인증 **또는** 구매안전서비스 비적용대상 확인서
- [ ] 사업자등록증 사본

## 함께 제출하는 서류

- 구매안전서비스 비적용대상에 해당하면 `구매안전서비스 비적용대상 확인서` 동시 제출

## 주요 유의사항

- 신고 후 **변경사항 발생 시 30일 이내** 변경 신고 의무 (§12③)
- 도메인 변경, 대표자 변경, 소재지 변경 모두 해당
- 폐업 시 폐업 신고 의무
