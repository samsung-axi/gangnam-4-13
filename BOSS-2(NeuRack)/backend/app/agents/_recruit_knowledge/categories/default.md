# 범용 채용 가이드 (업종 미지정)

## 공통 원칙

- **근기법 §17 필수 기재사항**이 공고에도 일부 반영돼야 신뢰도 상승
- 시급은 **최저임금(2026년 10,320원) 이상** 숫자로 표기
- 허위·과장 금지 (채용절차법 §4) — "고수익 보장" 금지
- 성별·연령·출신 차별 금지 (남녀고용평등법 §7)

## 공고에 반드시 포함

- 직종·포지션 명확히 (모호한 "스태프" 대신 구체 직무)
- 근무지 (지역구 + 지하철역 근처면 명시)
- 고용 형태 (정규/계약/파트/알바)
- 시급/월급/연봉 중 하나를 숫자로
- 주 근무시간 / 근무 요일 / 근무 시간대
- 모집 인원
- 지원 방법·연락처

## 공통 면접 질문

### 자기이해

- 이 공고를 보고 지원한 이유 한 가지?
- 본인의 강점 하나와 약점 하나?
- 다음 3개월 / 1년 목표는?

### 근무 태도

- 지각/결근이 불가피할 때 연락 매너?
- 팀원과 의견 충돌 시 해결 방식?
- 업무 실수를 상사에게 보고하는 방식?

### 지속 가능성

- 예상하는 근무 기간?
- 학업/다른 일과 병행 가능한 시간대?

## 피해야 할 문구

- "열정 있는 분" — 구체적이지 않음
- "가족같은 분위기" — 과도한 초과근무 신호로 읽힘
- "책임감 있는 분" — 당연한 요건, 차별화 없음
