# 소매/편의점/판매 채용 가이드

## 자주 뽑는 직무

- **캐셔/계산원** — POS 결제, 포장, 영수증 처리
- **매장 관리** — 재고·진열·청소·발주
- **상품 진열** — 신상품 입고·진열·유통기한 관리
- **야간 알바** — 편의점 24h 근무
- **점장/매니저** — 매출 관리·발주·근무표

## 시급 가이드 (2026년, 서울 기준)

- 편의점 주간 미경험: 10,320원
- 편의점 야간 (22~06시): 최저임금 + 야간 50% 가산 = 15,480원
- 의류/패션 매장: 11,000 ~ 13,000원 + 판매 인센티브
- 백화점/명품 매장: 12,500 ~ 16,000원 + 인센티브
- 점장 정규직: 연봉 3,200만 ~ 4,500만원

## 공고에 강조할 것

- **야간근무**는 반드시 야간수당 별도 명시 (근기법 §56)
- 도난 사고 시 **사업주 책임** 원칙 — "손해 배상" 문구 금지
- 인센티브가 있으면 지급 기준 공개 (허위 광고 금지)
- 서있는 시간 · 무거운 물건 운반 여부 미리 고지

## 면접 질문 뱅크

### 기본

- 저희 매장 방문 경험 있으신가요? 인상?
- 하루 6시간 서서 근무 가능하신가요?
- POS 사용 경험 있으신가요?

### 고객 응대

- 컴플레인 손님 대응 순서? (사과 → 경청 → 해결 → 재확인)
- 진상 손님과 다른 손님이 동시 대기 중이면 우선순위?
- 환불 규정을 손님이 받아들이지 않을 때 어떻게?

### 재고·현금

- 현금 시재가 맞지 않을 때 어떻게 추적하시나요?
- 유통기한 임박 상품 판매 전략?
- 도난 의심 시 증거 없이 고객 제지 가능한가요?

### 야간 근무 (편의점 특화)

- 야간 근무 시 가장 우려되는 상황은?
- 혼자 근무 중 위협적인 손님 등장 시 대처?
- 시간대별 방문 패턴 파악 경험 있으신가요?
