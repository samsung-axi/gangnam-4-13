# 알바천국 공고 베스트 프랙티스

## 톤 & 스타일

- **표준 구인 양식 + 모바일 친화** — 굵은 키워드, bullet 리스트
- 시급을 제목 가까이에 **반드시 노출**
- 기간·시간대를 숫자로 명확히

## 권장 구조

```
[공고 제목]
N,NNN원 / [지역 상권] [직종] 알바

[모집 내용]
• 업무: 카운터·음료 제조·마감
• 인원: N명
• 근무 기간: 단기(2주) / 장기(3개월+) / 협의
• 근무 요일: 월·화·수·목·금 (주말 휴무)
• 근무 시간: 09:00 ~ 14:00 (1일 5시간)

[급여 조건]
• 시급: N,NNN원 (2026년 최저임금 + α)
• 주휴수당: 포함 / 별도 지급
• 4대보험: 주 15h 이상 시 가입

[우대 사항]
• 바리스타 자격증 보유자
• 인근 거주자 (도보 10분 이내)
• 카페 경력 3개월 이상

[복리후생]
• 식사 제공 (점심 1식)
• 음료 무제한
• 명절 상여

[지원 방법]
• 알바천국 지원 버튼
• 문의: 010-XXXX-XXXX (오후 3~5시)
```

## 키워드 최적화

- 검색 빈도 높은 키워드: "단기", "장기", "주말알바", "오전알바", "마감알바"
- 지역명 + 역명 (예: "연남동 홍대입구역 3분")
- 직종 한글·영문 병기 ("바리스타 / Barista")

## 법정 고지 필수

- 최저임금 이상 시급 명시 (위반 시 광고 reject)
- "성별 무관" 기본 가정, 특별 사유 없이 남녀 구분 금지
- 나이 제한 — 만 18세 미만은 연소자 근로 특칙 고지

## 피해야 할 것

- 제목에 "고수익" 과장 표현
- "지원자 전원 면접" 같은 허위 문구
- 시급 범위를 지나치게 넓게 ("8,000~15,000원")
