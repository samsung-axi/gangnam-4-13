# 사람인 공고 베스트 프랙티스

## 톤 & 스타일

- **공식적·회사 브랜딩 중심** — 정규직/경력직 구직자 타겟
- 회사 소개 → 담당 업무 → 자격 요건 → 우대사항 → 복리후생 → 근무조건 → 전형 절차 순
- 존댓말·문어체. 이모지 사용 자제.

## 권장 구조

```
[공고 제목]
(주)○○커피 | 연남동 카페 직영점 홀 매니저 모집

[회사 소개]
2020년 연남동에서 시작한 스페셜티 카페로, 현재 마포구 2개 매장을 운영하고 있습니다.
에스프레소와 드립 커피를 중심으로 한 음료 품질과 고객 경험에 집중합니다.

[담당 업무]
• 매장 오퍼레이션 관리 (오픈·마감 셰프팅)
• 신규 직원 트레이닝
• 원두·소모품 발주 관리
• 월간 매출·지표 리포트 작성

[자격 요건]
• 카페 매니저 경력 2년 이상
• 주 5일 풀타임 근무 가능자
• 엑셀·구글시트 중급 이상

[우대 사항]
• 바리스타 2급 자격증
• SCA 브루잉 인증
• 매장 오픈 경험

[근무 조건]
• 고용 형태: 정규직 (수습 3개월)
• 근무지: 서울 마포구 연남동 ○○로 N길
• 근무 시간: 주 40시간 (월~금 09:00~18:00)
• 급여: 세전 연봉 3,200만원 ~ 3,800만원 (경력 협의)
• 4대보험 가입 / 퇴직연금 / 식대 지원 / 월 1회 팀 외식

[전형 절차]
1. 서류 전형 (이력서 + 자기소개서)
2. 1차 실무 면접 (30분)
3. 2차 매장 방문 테스트 (반나절)
4. 최종 합격 통지 (1주일 이내)

[지원 방법]
• 사람인 입사지원 시스템
• 문의: hr@example.com
```

## 필수 고지 사항

- 근기법 §17 필수 기재 (임금·근로시간·휴일·업무 내용·취업장소)
- 채용절차법 §10 — 합격/불합격 결과 통지 의무
- 성별·연령·출신지역·혼인 등 차별 금지 (남녀고용평등법 §7)

## 키워드 / 필터

- 직무 카테고리, 경력 연차, 학력, 지역, 고용형태 모두 메타데이터로 입력
- 연봉 범위는 숫자로 (회사 내부 규정상 비공개면 "회사내규"로 명시, 허위금지)

## 피해야 할 것

- 회사명 생략 (사람인은 기업회원 신원 확인이 강함)
- "열정 있는 분", "가족같은 분위기" 등 모호·감정 문구 → 신뢰도 저하
- 초과근무를 "업무 집중도 높은 분" 같은 식으로 포장
