# 🚀 배포 체크리스트

AWS Lightsail 배포 전 확인사항을 체크하세요.

## 📋 사전 준비

### AWS 리소스
- [ ] AWS 계정 생성 및 로그인
- [ ] Lightsail 인스턴스 생성 ($40/월 플랜 이상 권장)
- [ ] 고정 IP 생성 및 연결
- [ ] 방화벽 규칙 설정 (HTTP 80, HTTPS 443, SSH 22)
- [ ] SSH 키 다운로드 및 저장

### API 키 및 인증 정보
- [ ] GEMINI_API_KEY 준비
- [ ] GOOGLE_CLIENT_ID / GOOGLE_CLIENT_SECRET 준비
- [ ] JWT_SECRET_KEY 생성 (`openssl rand -hex 32`)
- [ ] PORTONE_API_KEY / PORTONE_API_SECRET 준비
- [ ] YOUTUBE_API_KEY 준비 (선택사항)
- [ ] TAVILY_API_KEY 준비 (선택사항)

### 도메인 (선택사항)
- [ ] 도메인 구매 또는 기존 도메인 준비
- [ ] DNS A 레코드 설정 계획
- [ ] SSL 인증서 발급 계획 (Let's Encrypt)

## 🐳 Docker 배포

### 빌드 및 실행
- [ ] 프론트엔드 빌드 완료 (`cd frontend && npm run build`)
- [ ] Docker 이미지 빌드 성공 (`docker-compose -f docker-compose.production.yml build`)
- [ ] 모든 컨테이너 시작 성공 (`docker-compose -f docker-compose.production.yml up -d`)
- [ ] 컨테이너 상태 확인 (모두 `Up` 상태)

### 서비스 확인
- [ ] MySQL 컨테이너 정상 작동
- [ ] FastAPI 컨테이너 정상 작동 (`curl http://localhost:8000/`)
- [ ] Nginx 컨테이너 정상 작동 (`curl http://localhost/`)
- [ ] VLM Worker 1 정상 작동
- [ ] VLM Worker 2 정상 작동

### 기능 테스트
- [ ] 프론트엔드 접속 가능 (`http://[서버-IP]/`)
- [ ] API 엔드포인트 접근 가능 (`http://[서버-IP]/api/`)
- [ ] 로그인 기능 테스트
- [ ] 비디오 업로드 기능 테스트
- [ ] HLS 스트리밍 기능 테스트

## 🔒 보안 설정

### 환경 변수 보안
- [ ] `.env.production` 파일이 `.gitignore`에 포함되어 있는지 확인
- [ ] 모든 비밀번호가 강력한지 확인 (최소 16자)
- [ ] `JWT_SECRET_KEY`가 안전하게 생성되었는지 확인
- [ ] 프로덕션 환경 변수가 개발 환경과 분리되어 있는지 확인

### 네트워크 보안
- [ ] MySQL 포트가 외부에 노출되지 않도록 설정 (127.0.0.1만 허용)
- [ ] FastAPI 포트가 외부에 노출되지 않도록 설정 (Nginx를 통해서만 접근)
- [ ] 방화벽 규칙이 최소 권한 원칙을 따르는지 확인
- [ ] SSH 키 인증만 허용하도록 설정 (비밀번호 인증 비활성화)

### SSL/TLS (선택사항)
- [ ] SSL 인증서 발급 완료 (Let's Encrypt)
- [ ] HTTPS 리다이렉션 설정 완료
- [ ] SSL 인증서 자동 갱신 설정 완료

## 🎯 배포 완료 후

배포가 완료되면 다음을 수행하세요:

1. **모니터링 시작**: 첫 24시간 동안 로그를 주의 깊게 모니터링
2. **사용자 테스트**: 실제 사용자에게 베타 테스트 요청
3. **성능 최적화**: 필요시 리소스 조정
4. **문서 업데이트**: 배포 중 발견된 문제점 문서화
