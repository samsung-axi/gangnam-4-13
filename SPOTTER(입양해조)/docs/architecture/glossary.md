# 도메인 용어집

이 프로젝트에서 사용하는 핵심 용어입니다. 코드, 주석, 변수명에서 일관되게 사용하세요.

| 한국어 | 영어 (코드용) | 설명 |
|--------|-------------|------|
| 상권 | commercial_area | 특정 지역의 소비 활동이 이뤄지는 범위 |
| 행정동 | district | 마포구 내 16개 행정구역 단위 (예: 망원1동, 공덕동) |
| 카니발리제이션 | cannibalization | 같은 브랜드 신규 매장이 기존 매장 매출을 잠식하는 현상 |
| 자기잠식률 | cannibalization_rate | 카니발리제이션으로 인한 매출 감소 비율 (0.0~1.0) |
| 유동인구 | floating_population | 특정 시간대에 해당 지역을 이동하는 인구 수 |
| 주거인구 | resident_population | 해당 지역에 거주하는 인구 수 |
| 손익분기점 (BEP) | bep_month | Break-Even Point. 누적 수익이 초기 투자를 회수하는 시점 |
| 직접 경쟁 | direct_competition | 동일 업종 내 경쟁 (예: 카페 vs 카페) |
| 간접 경쟁 | indirect_competition | 대체재와의 경쟁 (예: 카페 vs 배달 음료) |
| 경쟁 반경 | competition_radius | 경쟁 영향권 거리 (위험 500m / 주의 1km / 안전 1.5km) |
| 가맹사업법 | franchise_act | 가맹사업거래의 공정화에 관한 법률 |
| 상가임대차보호법 | commercial_lease_act | 상가건물 임대차보호법 |
| 출점 | store_opening | 신규 매장 개설 |
| 업종 | business_type | 사업 유형 (cafe, restaurant, convenience) |
| 소상공인 | small_business | SEMAS 기준 소규모 사업자 |
