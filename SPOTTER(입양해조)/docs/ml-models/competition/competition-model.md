# 경쟁 분석 모델 설계 문서

## 3-Layer 경쟁 분석 프레임워크

### Layer 1: 직접 경쟁 (Direct Competition)
- 동일 업종 매장과의 경쟁
- 반경별 매장 수 (500m / 1km / 1.5km)
- 포화도 지수 = 점포 수 / 유동인구 × 10,000

### Layer 2: 카니발리제이션 (Cannibalization)
- 동일 브랜드 기존 매장과의 자기 잠식
- 거리 기반 잠식률: 500m 이내(높음) → 1.5km 이상(독립)
- 본사 순증 매출 = 신규 매출 - Σ(기존 매장 잠식분)
- 가중치: 1.5 (가장 높음)

### Layer 3: 간접 경쟁 (Indirect/Substitute Competition)
- 대체재 카테고리와의 소비 예산 경쟁
- 예: 치킨집 경쟁상대 = 피자/족발/중식 등 배달 야식 전체
- 가중치: 0.5

## 경쟁 반경 기준

| 반경 | 위험도 | 설명 |
|------|--------|------|
| 500m 이내 | 위험 | 높은 자기 잠식 |
| 500m~1km | 주의 | 부분 잠식 |
| 1km 이상 | 안전 | 독립 상권 |
