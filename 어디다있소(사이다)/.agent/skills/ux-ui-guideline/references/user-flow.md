# User Flow (4 Stage)

> 아래 플로우차트를 함께 참고하세요.
> ![User Flow 다이어그램](../user-flow.png)

## Stage 1: 입력 및 의도 분석 (태블릿)

1. **시작** — 태블릿 키오스크 실행, 홈 화면 표시
2. **상품검색** — 음성 또는 텍스트로 상품명/키워드 입력
3. **의도분석** — 입력이 "상품 위치 찾기" 의도인지 판별
   - **Yes** → 4번으로 진행
   - **No** → 위치 안내 불가 안내 (질문 예시 재안내 또는 직원 호출) → 2번으로 복귀
4. **키워드 추출** — 검색 가능한 상품 키워드로 정제

## Stage 2: 검색 (태블릿)

5. **검색 실행** — 추출된 키워드로 상품 DB 검색
6. **검색 결과 판별** — 결과 유무 및 개수 확인
7. **결과 화면 출력** — 결과 있음: 상품 카드 리스트(최대 3개) + 매장 지도 표시 / 결과 없음: "찾을 수 없습니다" + 재검색 유도

## Stage 3: 결과 안내 (태블릿)

8. **상품 선택** — 사용자가 원하는 상품 카드 탭
9. **매대 번호 및 QR 코드 안내** — 선택 상품의 위치코드(예: B1-B01) + QR 코드 표시
   - 사용자가 "원하는 상품 없음" 선택 → 13번으로
13. **태블릿 종료** — 일정 시간 후 자동 또는 다음 사용자 터치 시 홈으로 복귀

## Stage 4: 모바일 연동 (모바일)

10. **모바일 카메라/QR 리더 실행** — 사용자 개인 스마트폰에서 카메라 앱 또는 QR 리더 실행
11. **QR 스캔** — 태블릿에 표시된 QR 코드 스캔 → 모바일 웹 페이지 오픈
12-1. **지도 보며 이동** — 매장 평면도 + 경로 점선으로 안내
12-2. **AR 네비 보며 이동** — 카메라 뷰 위 AR 화살표로 안내 (시뮬레이션)
14. **모바일 안내 종료** — 목적지 도착 또는 사용자가 종료
