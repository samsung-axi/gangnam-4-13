# 매장 경로 탐색 데이터 (Map Pathfinding)

이 문서는 매장 평면도 위에 경로를 시각화하기 위한 waypoint 그래프 데이터입니다.
경로 탐색 알고리즘(BFS/Dijkstra)에서 이 데이터를 사용합니다.

- 좌표는 이미지 좌상단 (0,0) 기준 백분율(%)
- 엣지는 양방향(↔)
- 카테고리 매핑은 해당 매대에 가장 가까운 노드를 지정

## 층간 연결 (Cross-Floor)

B1↔B2 이동 시 아래 가상 엣지를 추가합니다.
층간 이동은 도보 이동보다 비용이 크므로 weight를 높게 설정합니다.

b1-entrance ↔ b2-entrance (weight: 5)

경로 탐색 시 "B1 식품 → B2 주방"처럼 층을 넘나드는 검색도 이 엣지를 통해 처리됩니다.
